To use this component, you need to install the `react-native-reanimated` library. Follow the steps below:

### Step 2: Install [`Hyper-Native-Ui`](https://www.npmjs.com/package/hyper-native-ui)

* Open a Terminal in your project's folder and run: `hyper-native-ui`

## Props

### `imageUrl`
- **Type:** `string`
- **Required:** ✅
- **Description:** The URL of the image used in the parallax header.

### `imageHeight`
- **Type:** `number`
- **Default:** `500`
- **Description:** The height of the parallax image header.

### `headerComponent`
- **Type:** `ReactNode`
- **Required:** ❌
- **Description:** Custom component to replace the default image header.

### `imageStyle`
- **Type:** `ImageStyle`
- **Required:** ❌
- **Description:** Custom styling for the image.

### `imageProps`
- **Type:** `ImageProps`
- **Required:** ❌
- **Description:** Additional props to pass to the `Image` component.

### `childrenStyle`
- **Type:** `ViewStyle`
- **Required:** ❌
- **Description:** Custom styling for the children container.

## Animated Behavior

The parallax effect is achieved using `react-native-reanimated`. The image smoothly scales and translates as the user scrolls, providing a visually appealing effect.

## Notes
- The component is styled using the theme context from `useTheme()`.
- The `scrollOffset` is managed using `useScrollViewOffset` from `react-native-reanimated`.

## Summary
The `ParallaxScrollView` is a flexible and customizable component that enhances user experience with smooth scrolling and animated parallax effects. It supports various props for customization, allowing you to create stunning UI layouts effortlessly.
