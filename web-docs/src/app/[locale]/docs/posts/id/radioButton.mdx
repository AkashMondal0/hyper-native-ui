The `RadioButton` component provides a visually appealing and interactive radio button group, allowing users to select one option from a list. It supports animations, theming, and various customizations.

### Step 1: Install [`React-Native-Reanimated`](https://docs.swmansion.com/react-native-reanimated/docs/fundamentals/getting-started/)

To use this component, you need to install the `react-native-reanimated` library. Follow the steps below:

### Step 2: Install [`Hyper-Native-Ui`](https://www.npmjs.com/package/hyper-native-ui)

* Open a Terminal in your project's folder and run: `hyper-native-ui`

### RadioButton Props

- **`options`**: `Option[]`  
  Array of option objects for the radio buttons.  

- **`selectedValue`**: `string | number`  
  The currently selected value.

- **`onSelect`**: `(value: string | number) => void`  
  Callback function triggered when a selection is made.

- **`size`**: `number`  
  Diameter of the radio button circle. Default: `24`.

- **`themeScheme`**: `"light" | "dark"`  
  Overrides the default theme scheme.

- **`variant`**: `"default" | "secondary" | ThemeName`  
  Sets the color variant of the radio buttons. Default: `"default"`.

- **`animationDuration`**: `number`  
  Duration of the animation in milliseconds. Default: `300`.

- **`radioOptionTextStyle`**: `TextStyle`  
  Custom text style for the radio option labels.

### Option Type

- **`label`**: `string`  
  The text displayed alongside the radio button.

- **`value`**: `string | number`  
  The value of the radio button.

- **`disabled`**: `boolean`  
  If true, the option is not selectable.

### RadioOption Props

- **`label`**: `string`  
  The text displayed alongside the radio button.

- **`value`**: `string | number`  
  The value of the radio button.

- **`isSelected`**: `boolean`  
  Determines if the radio button is currently selected.

- **`onPress`**: `(value: string | number) => void`  
  Function triggered when the radio button is pressed.

- **`size`**: `number`  
  Diameter of the radio button circle.

- **`color`**: `string`  
  Color of the selected radio button.

- **`textColor`**: `string`  
  Text color for the label.

- **`disabled`**: `boolean`  
  Indicates if the option is not selectable.

- **`radioOptionTextStyle`**: `TextStyle`  
  Custom text style for the radio option labels.
