### Required Props

None. All props are optional and designed for flexibility.

### Optional Props

- **`variant`** (`string`): Defines the color scheme of the component. Defaults to `default`.
- **`center`** (`boolean`): Aligns the component horizontally. Defaults to `false`.
- **`width`** (`number`): Sets the width of the component.
- **`height`** (`number`): Sets the height of the component.
- **`themeScheme`** (`object`): Overrides the global theme with custom colors.

## Styling and Behavior

- **Theme Integration**:
  - The component adapts to the current theme, applying `primary`, `secondary`, and other defined colors based on the `variant` prop.
  - `themeScheme` allows overriding the global theme.

- **Alignment**:
  - Use the `center` prop for horizontal alignment.

- **Custom Dimensions**:
  - Define `width` and `height` for specific layouts.

- **Variants**:
  - **`default`**: Primary color from the theme.
  - **`secondary`**: Secondary color scheme.
  - **`danger`**: Error or destructive style.
  - **`warning`**: Caution or alert style.
  - **`success`**: Indicates success.
  - Extendable with custom themes.
