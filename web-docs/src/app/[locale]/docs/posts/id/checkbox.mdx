The `CheckBox` component is a flexible and customizable checkbox built on top of [react-native-bouncy-checkbox](https://www.npmjs.com/package/react-native-bouncy-checkbox) .
 It supports a variety of themes and variants, allowing it to integrate seamlessly into diverse design systems.
 
### Step 1: Install [`Hyper-Native-Ui`](https://www.npmjs.com/package/hyper-native-ui)

## Props

### `themeScheme`

- **Type**: `"light" | "dark"`
- **Optional**
- **Description**: Specifies the theme scheme for the checkbox. If not provided, it defaults to the current theme scheme retrieved from the `useTheme` hook.

### `variant`

- **Type**: `"default" | "secondary" | "danger" | "warning" | "success" | "outline" | "Zinc" | "State" | "Stone" | "Grey" | "Neutral" | "Red" | "Rose" | "Orange" | "Yellow" | "Green"`
- **Optional**
- **Default**: `"default"`
- **Description**: Specifies the visual style variant of the checkbox. Custom theme colors are applied based on the provided `variant`.

### `size`

- **Type**: `number`
- **Optional**
- **Default**: `25`
- **Description**: Specifies the size of the checkbox (in pixels).

### `...props`

- **Type**: `BouncyCheckboxProps`
- **Optional**
- **Description**: Additional props to be passed down to the `BouncyCheckbox` component. Refer to the `react-native-bouncy-checkbox` documentation for a list of all supported props.

## How It Works

The component determines its appearance based on the provided `variant` and `themeScheme`. If a variant matches a predefined theme name, the corresponding colors are applied. Otherwise, it falls back to the `default` styling.

The `useTheme` hook provides the `currentTheme` and `defaultThemeScheme` for seamless integration with your app's global theme.

### Theme Color Resolution Logic

1. If the `variant` is `"default"` or `"secondary"`, the current theme colors are applied.
2. If a custom `variant` (e.g., `"Red"`, `"Orange"`) is provided, the component uses matching theme colors from `themeColors`.
3. Fallback colors are applied when no matching theme or variant is found.
