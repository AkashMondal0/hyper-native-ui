The `TouchableOpacity` is a customizable and theme-aware wrapper for React Native's `TouchableOpacity`. It changes its background color dynamically when pressed, based on the app's current theme.

### Step 1: Install [`Hyper-Native-Ui`](https://www.npmjs.com/package/hyper-native-ui)

* Open a Terminal in your project's folder and run: `hyper-native-ui`

### Custom Props

- **`variant`** (`any`): Placeholder for future variants or styles. Currently unused.
- **`lightColor`** (`string`): A color to use in light mode (optional).
- **`darkColor`** (`string`): A color to use in dark mode (optional).

### All TouchableOpacity Props

This component also accepts all props that can be passed to React Native's `TouchableOpacity`. Refer to the [TouchableOpacity documentation](https://reactnative.dev/docs/touchableopacity) for details.

## Features

1. **Theme-Aware Styling**:
   - Automatically adjusts background color based on the active theme.
   - Supports custom light and dark colors via the `lightColor` and `darkColor` props.

2. **Press Feedback**:
   - Changes the background color when pressed, providing visual feedback.

3. **Customizable**:
   - Fully compatible with `TouchableOpacity` props, allowing for advanced usage.

4. **Delay on Press**:
   - Includes a slight delay (`delayPressIn`) to ensure smooth transitions during press actions.
