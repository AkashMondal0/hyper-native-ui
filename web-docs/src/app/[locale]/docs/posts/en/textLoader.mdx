`TextLoader` is a React Native component that animates text as if it is being typed out, with a pulsing ellipsis at the end. It supports different text variants, colors, and speed adjustments.

### Step 1: Install [`Hyper-Native-Ui`](https://www.npmjs.com/package/hyper-native-ui)

* Open a Terminal in your project's folder and run: `hyper-native-ui`

### Props
- `text`: `string` - The text to be animated. Default is `"Generating response..."`.
- `speed`: `number` - Speed of the typing effect in milliseconds per character. Default is `30`.
- `onComplete`: `() => void` - Callback function when typing is completed. Default is `undefined`.
- `themeScheme`: `'light' | 'dark'` - Theme scheme for styling. Default is `undefined`.
- `variantColor`: `'default' | 'primary' | 'secondary' | ThemeName` - Color variant of the text. Default is `'default'`.
- `variant`: Various text variants (H1, H2, etc.) - Typography variant. Default is `'body1'`.
- `bold`: `TextStyle['fontWeight']` - Custom font weight. Default is `undefined`.
- `size`: `TextStyle['fontSize']` - Custom font size. Default is `undefined`.
- `center`: `boolean` - Whether text is centered. Default is `false`.
- `containerStyle`: `ViewProps` - Style for the text container. Default is `undefined`.
