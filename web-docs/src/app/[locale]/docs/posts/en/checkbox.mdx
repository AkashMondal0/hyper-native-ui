The `Checkbox` component is a customizable, theme-aware, and animated checkbox for React Native. It supports dynamic styling, bounce animations, and various configuration options to suit your app's design and functionality.

To use this component, you need to install the `react-native-reanimated` library. Follow the steps below:

### Step 2: Install [`Hyper-Native-Ui`](https://www.npmjs.com/package/hyper-native-ui)

* Open a Terminal in your project's folder and run: `hyper-native-ui`

### Custom Props

- **`label`** (`string`): The text displayed next to the checkbox.
- **`isSelected`** (`boolean`, **required**): Indicates whether the checkbox is selected or not.
- **`onToggle`** (`(isSelected: boolean) => void`, **required**): A callback function that is triggered when the checkbox is toggled.
- **`size`** (`number`, default: `24`): Specifies the width and height of the checkbox in pixels.
- **`bounceIntensity`** (`number`, default: `8`): Controls the intensity of the bounce animation when toggled.
- **`themeScheme`** (`"light" | "dark"`): Allows overriding the theme scheme for the component.
- **`variant`** (`"default" | "secondary" | ThemeName`, default: `"default"`): Determines the color theme for the checkbox.
- **`innerComponent`** (`ReactNode`): A custom component rendered inside the checkbox when selected.
- **`disabled`** (`boolean`): Disables the checkbox, making it unclickable and dimmed.
- **`radius`** (`number`): Adjusts the border radius of the checkbox. Defaults to `size / 4` if not provided.
- **`labelTextStyle`** (`TextStyle`): Customizes the style of the label text.

### Animated Properties

The checkbox uses React Native Reanimated for smooth transitions and animations. These include:

- **Scale**: The checkbox bounces when toggled.
- **Border Width**: The border smoothly transitions based on the selection state.

### Theming

The component relies on the `useTheme` hook for theme-awareness. It adapts its colors dynamically based on the current theme.

## Features

1. **Dynamic Styling**:
   - Adjusts colors and borders according to the selected theme or provided props.
2. **Bounce Animation**:
   - Adds a spring animation effect when toggling.
3. **Customizable**:
   - Supports custom components, sizes, radii, and labels.
4. **Theme-Aware**:
   - Automatically adjusts appearance based on light or dark mode.
5. **Disabled State**:
   - Dims the component and disables interaction when `disabled` is `true`.
