The `Dropdown` component is a customizable and animated dropdown menu built with React Native and Reanimated. It allows users to select an item from a list with smooth animation effects and theming support.

## Installation

* Open a Terminal in your project's folder and run:

### Optional Props

- **`data`** (`DropdownItem[]`, default: `[]`): Array of dropdown items where each item is an object with `label` and `value`.
- **`placeholder`** (`string`, default: `"Select an option"`): Placeholder text displayed when no item is selected.
- **`onSelect`** (`(item: DropdownItem) => void`, default: `undefined`): Callback function invoked when an item is selected.
- **`dropdownStyle`** (`ViewStyle`, default: `{}`): Custom styles for the dropdown container.
- **`itemStyle`** (`ViewStyle`, default: `{}`): Custom styles for each dropdown item.
- **`itemTextStyle`** (`TextStyle`, default: `{}`): Custom styles for the text inside each dropdown item.
- **`dropdownAnimationDuration`** (`number`, default: `300`): Duration of the dropdown expand/collapse animation in milliseconds.

## Behavior

1. **Dropdown Animation**:
   - Uses `react-native-reanimated` for smooth height-based animations.
   - The dropdown expands and collapses with customizable animation duration.

2. **Dynamic Selection**:
   - Displays a placeholder text until an item is selected.
   - Updates the displayed text upon item selection.

3. **Customization**:
   - Customizable dropdown styles, item styles, and text styles.
   - Supports themes via the `useTheme` hook.

4. **Accessibility**:
   - Automatically adjusts the list's height and visibility for a smooth user experience.
