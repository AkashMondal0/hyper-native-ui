## Usage

The `ButtonExample` function is a React Native component that demonstrates the usage of a button to toggle between themes (e.g., light and dark modes) in an application.

### **Purpose**  
This component provides an interactive button that switches the app's theme when pressed. It leverages the `useTheme` custom hook to retrieve the current theme and toggle it dynamically.

### **Functionality**  

- **Theme Management**  
   - The `useTheme` hook exposes `currentTheme` and `toggleTheme`.
     - `currentTheme`: Provides theme-specific properties, including the background color.  
     - `toggleTheme`: A function that switches the app's theme (e.g., light to dark or vice versa).

- **UI Layout**  
   - The component uses a centered `View` with a background color based on the `currentTheme`.  
   - A button is displayed at the center of the screen.

- **Button Interaction**  
   - When the button is pressed, the `toggleTheme` function is triggered, which switches the theme.
